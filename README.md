## 📋 About

Maze Group is a B2B supplier specializing in hotel equipment and IT infrastructure solutions for the hospitality industry across the South Caucasus and CIS regions. This repository contains the full source code for the company's official website, built with modern web technologies and designed for performance, SEO, and multilingual support.

---

## ✨ Features

- 🌍 **Multilingual support** — internationalization with `next-intl` (i18n routing)
- ⚡ **Next.js App Router** — fast, SEO-friendly server-side rendering
- 🎨 **Fully responsive** — optimized for all screen sizes
- 🔒 **TypeScript** — fully typed codebase (99.5% TypeScript)
- 📦 **Modular architecture** — clean separation of components, services, types & lib
- 🚀 **Deployed on Vercel** — CI/CD with automatic deployments

---

## 🛠 Tech Stack

| Technology | Usage |

## 📁 Project Structure

```
├── app/              # Next.js App Router pages
├── components/       # Reusable UI components
├── messages/         # i18n translation files
├── services/         # API & data services
├── types/            # TypeScript type definitions
├── lib/              # Utility functions & helpers
├── assets/           # Static assets
├── public/           # Public files
└── middleware.ts     # i18n locale middleware
```

---



## 🌍 Internationalization

The site supports multiple languages via `next-intl`. Translation files are located in the `/messages` directory. Locale routing is handled by `middleware.ts`.
